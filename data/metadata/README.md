# Metadata Directory

This directory stores benchmark metadata tables used across the AlphaGenome phase 3 QTL evaluation pipeline.

- `sample_metadata.tsv`: Minimal per-dataset sample descriptors used to track assay type and source references.
- `tissue_mappings.tsv`: Manual and ontology-based mappings between dataset cell types and AlphaGenome ontology labels.

Add additional metadata files as the benchmark expands (e.g., phenotype ontologies, QC metrics, biosample harmonization tables).
